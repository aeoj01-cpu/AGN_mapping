# AGN Mapping Project

This project investigates Active Galactic Nuclei (AGN) properties and spatial distributions using astronomical survey data.

## Project Structure

```
AGN_mapping/
│
├── data/        # Raw and processed datasets
├── notebooks/   # Jupyter analysis notebooks
├── src/         # Reusable Python modules
├── figures/     # Generated plots
├── environment.yml
└── README.md
```

## Setup

Create the environment:

```
conda env create -f environment.yml
conda activate agn
```
